Getting started

Template files

The first step would be to get familiarized with the zip file you have downloaded from ThemeForest.

  • images - Here goes all stuff like site images, weather icons...
  • demo - Demo content like images, logo... (you can remove this)
  • fonts - Fonts files (Fontawesome and Socicon)
  • css - All css files for website
  • js - All javascript (jQuery) used on site.

Wide/boxed

Wide

Wide layout is by default, it means that you don't need to add any class to body or wrapper.

Boxed

To set your boxed layout you need to add class .boxed to div wrapper. You can set your background color or background image in style.css at line 17 body {}

<div id="wrapper" class="boxed">

Colors

Primary

Base color is blue, but you can easy set your own color for background, text and border in colors.css.

Header

Top bar

Top bar can be dark or light.

<div class="top-bar dark">
<div class="top-bar light">

Weather report

There are 8 states of weather. All have light and dark image. So if you set to light bar you will need to include dark image and vice verse. All weather images are placed into images/weather folder. Prefix of images are light-* or dark-*

<div class="weather-report">
    <span class="report">+28°C</span>
    <img src="images/weather/dark-weather-thunder.png" alt="Sunny">
    <span class="city">London, England</span>
</div>
<div class="weather-report">
    <span class="report">+28°C</span>
    <img src="images/weather/light-weather-thunder.png" alt="Sunny">
    <span class="city">London, England</span>
</div>

Header

Container with logo and banner can be dark or light.

<header id="header" class="dark" role="banner">
<header id="header" class="light" role="banner">

Logo

Logo can be textual or with image linked to your index page. Here is the example with text and image.

<div id="logo">
    <h1 id="site-logo"><a href="index.html">Dialy<span>Magazine</span></a></h1>
    <h2 id="site-description">Premium HTML5 Magazine Theme</h2>
</div>
<div id="logo">
    <a href="index.html"><img src="demo/DialyMagazine.png"/></a>
    <h2 id="site-description">Premium HTML5 Magazine Theme</h2>
</div>

Main menu

Primary navigation can be dark or light.

<nav class="primary-menu dark sticky-menu" role="navigation">
<nav class="primary-menu light sticky-menu" role="navigation">

Fixed navigation

Enable/disable

You can enable or disable fixed nav in your site by adding class .sticky-menu. By removing this class you will get none fixed navigation.

<nav class="primary-menu dark sticky-menu" role="navigation">

Fonts family

Body

Body use Google font Open Sans. You can change it by adding link from Google Fonts library. In head on each page you replace <link href='http://fonts.googleapis.com/css?family=Open+Sans:300italic,400italic,700italic,400,300,700' rel='stylesheet' type='text/css'> for your own font.

After that open your fonts.css from css folder and replace css rule with your own font family.

Headings

Headings and few more tags use Google font Roboto Slab. You can change it by adding link from Google Fonts library. In head on each page you replace <link href='http://fonts.googleapis.com/css?family=Roboto+Slab:400,100,300,700' rel='stylesheet' type='text/css'> for your own font.

After that open your fonts.css from css folder and replace css rule with your own font family.

Contact form

Email

You get fully functional contact form. You need to set your email address in contact.php page at line 3 and subject title in line 2 at same page.

$subjectPrefix = '[Write the title for email header]';
$emailTo = '[Write your email here]';

Above the fold

Slider and two posts

Above the fold is space for slider, grid posts and page title. In index.html you will find slider + 2 last posts. Slider can have any number of slides.

Grid posts

In index_1.html you will find grid posts. In demo there are 2 rows with 6 posts. You can have as mayn rows as you want. But needs to be filed with posts. Every row has 3 posts.

Page title

In shop.html you will find page title.

Category title

In blog_grid.html you will find category title with custom text color. This color is the same as blog catgeory and menu border in main menu.
